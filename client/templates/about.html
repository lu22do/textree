Overview

This website allows to create and read text trees, which are texts organized in an inverted tree-like structure. See Help for more details on how it works.

I had the first ideas about this project around 2000 but it is only in 2013 when I bumped into Node.js and its magic world that I finally realized I might to able to build it myslef.

Open source

This is an open source project. Source code of the client/server can be found on github @ github.com/lu22do/textree.
